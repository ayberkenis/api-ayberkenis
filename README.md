# api-ayberkenis
**You can visit [api.ayberkenis.online](api.ayberkenis.online) to be able to see API endpoints and results.**

- Earthquakes API

> All Earthquakes Endpoint: https://api.ayberkenis.online/earthquake/all

_No arguments needed._

`Return type: JSON`

> Latest Earthquake Endpoint: https://api.ayberkenis.online/earthquake/latest

_No arguments needed._

`Return type: JSON`

> Query an Earthquake Endpoint: https://api.ayberkenis.online/earthquakes/q

_Available arguments:_

  - time
  - date
  - latitude
  - longitude
  - depth
  - magnitude_md
  - magnitude_ml
  - magnitude_mw
  - location
  - state
  - google_maps

`Return type: JSON`

*** 

- Exchanges API

> All Exchanges Endpoint: https://api.ayberkenis.online/exchange/all

_No arguments needed._

`Return type: JSON`

> Query an Exchange Endpoint: https://api.ayberkenis.online/exchange/q

_Available arguments:_

  - currency_name
  - currency_code
  - currency_buy
  - currency_sell
  - currency_high
  - currency_low
  - currency_change
  - time

`Return type: JSON`
