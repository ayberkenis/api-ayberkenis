Earthquakes API

  • All Earthquakes Endpoint: https://api.ayberkenis.online/earthquake/all

    No arguments needed.

    Return type: JSON

  • Latest Earthquake Endpoint: https://api.ayberkenis.online/earthquake/latest

    No arguments needed.

    Return type: JSON

  • Query an Earthquake Endpoint: https://api.ayberkenis.online/earthquakes/q

    Available arguments:

    • time
    • date
    • latitude
    • longitude
    • depth
    • magnitude_md
    • magnitude_ml
    • magnitude_mw
    • location
    • state
    • google_maps

    Return type: JSON

Exchanges API

  • All Exchanges Endpoint: https://api.ayberkenis.online/exchange/all

    No arguments needed.

    Return type: JSON

  • Query an Exchange Endpoint: https://api.ayberkenis.online/exchange/q

    Available arguments:

    • currency_name
    • currency_code
    • currency_buy
    • currency_sell
    • currency_high
    • currency_low
    • currency_change
    • time

    Return type: JSON
